Escritura digital

Es la manera más básica de interactuar con módulos y componentes que hay en Arduino. Sólo podemos hacer referencia a dos estados: encendido o apagado.

Encendido Apagado
Activado Desactivado
5 volts 0 volts
HIGH LOW
true false
1 0

Función pinMode()

Normalmente utilizado en la función setup() para configurar uno a uno todos los pines digitales, indicándole a la placa si estos serán de entrada (escritura) o de salida (lectura).

Sintaxis

pinMode(int pin, modo)

Parámetros

  • pin: pin a configurar.
  • modo: INPUT si es de entrada, OUTPUT si es de salida.

Existe también un modo llamado INPUT_PULLUP que invierte los valores del INPUT normal.

Función digitalWrite()

Es la manera esencial de realizar una escritura digital por uno de los pines ya configurados previamente mediante la función pinMode().

Sintaxis

digitalWrite(int pin, valor)

Parámetros

  • pin: pin donde se hará la escritura digital.
  • valor: HIGH para encendido (5V), LOW para apagado (0V).

Existe también un modo llamado INPUT_PULLUP que invierte los valores del INPUT normal.

Función delay()

Pausa la ejecución del programa por un tiempo determinado, normalmente se utiliza en el loop().

Sintaxis

delay(long milisegundos)

Parámetro

  • milisegundos: valor en milésimas de segundos que se pausará el programa.

Parpadear un led

Para eso nos vamos a: Archivo > Ejemplos > 01.Basics > Blink

Constante LED_BUILTIN

Es una constante definida por defecto en el IDE de Arduino, corresponde al pin digital 13 de Arduino.

Si se reemplazara todos los LED_BUILTIN por 13 el programa funcionaría de la misma forma.

Modificando el código

Se eliminarán los comentarios para escribir unos propios, se reemplazarán todos los LED_BUILTIN por 13, también se variarán los valores de los parámetros en las funciones delay() a gusto. Luego de las modificaciones, el código nos debería quedar algo así:

void setup() {
  pinMode(13, OUTPUT); // Se configura el pin 13 (LED_BUILTIN) como salida 
}

void loop() {
  digitalWrite(13, HIGH); // Envia 5V al pin 13 (Enciende el led)
  delay(3000); // Espera 3000 milisegudos = 3 segundo
  digitalWrite(13, LOW); // Deja de enviar corriente (0V) al pin 13 (Apaga el led)
  delay(500); // Espera 500 milisegudos = 1/2 de segundo 
}

Ahora el tiempo que el led se mantiene encendido o apagado ha variado un poco.
