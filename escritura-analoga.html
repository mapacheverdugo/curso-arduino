Escritura análoga

PMW

Función analogWrite()

Sintaxis

analogWrite(int pin, int valor)

Parámetros

  • pin: pin donde se hará la escritura análoga.
  • valor: valor del ciclo de trabajo desde el 0 al 255.

Donde 0 es siempre apagado y 255 es siempre encendido.

Función tone()

Sintaxis

tone(int pin, int frecuencia)

tone(int pin, int frecuencia, int duracion)

Parámetros

  • pin: pin donde se hará el tono.
  • frecuencia: la frecuencia del tono en hertz.
  • duracion: duración del tono en milisegundos (opcional).
